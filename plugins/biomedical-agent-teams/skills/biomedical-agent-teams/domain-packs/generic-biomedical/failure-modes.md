# Generic Biomedical Failure Modes

- ID drift: PMID, DOI, accession, trial ID, or software version is stale or fabricated.
- Scope drift: species, cell type, assay, endpoint, or disease context differs from the final claim.
- Causal overclaim: correlation, descriptive omics, or low-quality evidence is written as mechanism.
- Clinical overreach: research evidence is written as treatment guidance without clinician review.
- Tool-use overclaim: the final answer says a database or validator was used without a successful tool-call ledger row.
